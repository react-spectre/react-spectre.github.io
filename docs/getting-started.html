Getting Started

React Spectre is a component library that provides a responsive and modern CSS framework for faster and extensible development based on Spectre.css.

Installation

  1. Get Spectre css files.

  2. Install one or more React Spectre packages with your package manager.

    npm install --save @react-spectre/<package-name>

    yarn add @react-spectre/<package-name>

Packages

React Spectre provides scoped packages for each Spectre.css elements. Also, for convenience it provide package set that group scoped packages as follow.
